# PIC24 CPU Implementation (VHDL)
Implementation of a PIC24-like CPU using VHDL.
## Features
- ALU operations (ADD, SUB, XOR, AND)
- Register file
- Control logic
- Program counter update
  ## Technologies
- VHDL
- ModelSim
- Xilinx ISE
  ## Description
This project implements a simplified PIC24 processor.
It includes core components such as the ALU, control unit and register file, simulated using hardware design tools.
